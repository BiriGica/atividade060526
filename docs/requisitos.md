## Requisitos

### RF01 — Cadastro de filmes
O sistema deve permitir cadastrar novos filmes contendo as informações epecíficos de cada um, como título e duração.

### RF02 — Organização de cada ssessão
O sistema deve permitir a criação e a organização das sessões, com base no filme em cartaz, um horário e a uma unidade de cinema da rede.

### RF03 — Controle de venda de ingressos
O sistema deve fornecer uma interface para conta da da quantidade exata de espectadores que compareceram a cada sessão realizada, contando quantos ingressos foram vendidos no dia.

### RF04 — Controle dos cinemas da rede
O sistema deve permitir o cadastro das unidades de cinema da rede.
